在electron中使用vue

创建vue项目

vue create electron-demo

选择自定义

选择配置

选择vue3

选择路由模式

后续操作就一路默认

创建完后，进入到项目 执行

npm run serve

然后访问

添加 electron支持

vue add electron-builder

然后执行

vue-cli-service electron:serve

就可以打开带有窗口的vue项目
